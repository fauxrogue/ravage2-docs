## 3. Install the addon

<!-- TODO: Add visual aid to the installation steps -->

Ensure that you have uninstalled any previous version of Ravage and restarted
Blender.

- Open the `Add-ons` tab in Blender's preferences

- Click on the `Install` button at the top right of the preferences panel

- Locate the `.zip` file

- Click on `Install Addon`

- Activate the addon

---

### If the installation fails, reopen Blender and follow the steps above again.
